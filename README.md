# Estudo-PR3C
O problema restrito de três corpos modela diversos sistemas na astrodinâmica. Esse projeto é um estudo da matemática envolvida e dos resultados obtidos pela sua aplicação. Vindo de um projeto de iniciação científica, o programa propaga órbitas, calcula os pontos de equilíbrio de lagrange, as curvas de velocidade zero e exibe todas as informações em gráficos.
